Market Summary Report (BI Reporting)

The Market Summary Report can be accessed from the Reporting tab in the Web UI.

This report allows you to compare the occupancy and rents for your properties and your competitors in different ways. It is useful for helping you spot negative trends quickly, with the added ability to drill down into the data for detailed analysis.

What questions can the Market Summary Report answer?

Viewing the Report

There are two views of the report, Overview and Detail, plus a Definitions page. You can switch between them using the tabs at the bottom of the screen.

The Overview and Detail view contain much of the same information. It is just displayed and filtered in a different way.

Community Information

Community Name (ID): Name and ID of the property

My Unit Type: LRO unit type

My Unit Type Description: Description of the unit type

Total Units: Total number of units in the unit type

SQFT: Square foot measurement of the unit type

Unit Type, Unit Type Description, Total Units, and SQFT are set up under Configuration>Unit Types.

Base Rent: Average rent without concessions for the unit type

Conc: Average concession amount for the unit type

Eff Rent: Average rent including concessions for the unit type (This will match the base rent if there are no concessions.)

Eff Rent/SQFT: Effective rent per square foot for the unit type

Competitor Information

Occupancy %: Percentage of units in this unit type that are occupied at the competitor property

My Unit Type: LRO unit type that is mapped to the competitor unit type

Their Unit Type Description: The competitor unit type

Total Units: Number of units the competitor has in the unit type

SQFT: Square foot measurement of the competitor unit type

Competitor Name (ID), mapping of unit types (My Unit Type), Their Unit Type Description, Total Units, and SQFT are entered on the Comp Setup screen.

Base Rent: Rent for the competitor unit type excluding any concessions

Conc: Concessions for the competitor unit type

Eff Rent: Effective rent for the competitor unit type (This will match the base rent if there are no concessions.)

Occupancy %, Base Rent, Conc, and Eff Rent are entered on the Comp Rents screen.

Eff Rent/SQFT: Effective rent per square foot for the competitor unit type

Weight: Indicates the relative importance of this competitor and is assigned on the Comp Weights screen (LRO>Competitors>Comp Weights)

Position: Indicates how your unit type compares with this competitor's unit type and is assigned on the Comp Positions screen (LRO>Competitors>Comp Positions)

Overview

When you open the Market Summary report, you'll be taken to the Overview. This view allows you to compare communities based on number of units (size of box) and effective rent (color of box). Typically, there will be 5 colors on the Overview. Communities with similar effective rents will be the same color. An attempt is made to group the communities so each color has about the same number of communities.

You can click on a community to drill down to the details for that community. When you drill down, you'll be able to see:

  • a pie chart with unit count by unit type for the community
  • a bar graph which allows you to compare the community's occupancy with that of its competitors
  • a bar graph which displays the average effective rent/square foot by unit type for the community
  • a bar graph which allows you to compare the community's average effective rent with the average effective rent of its competitors

Detail View

The Detail View of the Market Summary report has information about your community's units and the community's competitor's units in a table format. This view allows you to compare your community's unit count and rent information by unit type to your competitors' unit count and rent information by unit type.
