Expiring Leases Report

The Expiring Leases Report is opened from the Detail View of the Renewals Report or from the Expiring Leases section of the Portfolio Trends Dashboard. It gives you a way to get the details behind the information that is presented on those reports.

Opening the Report

To open the Expiring Leases Report from the Renewals Report:

  1. On the Detail View, select Hierarchy & Community or Hierarchy, Community & Unit Type as the Level to Display.

  2. Click any of the renewal numbers for the desired community or community and unit type to view the leases that make up those numbers.

To open the Expiring Leases Report from the Portfolio Trends Dashboard, click on the Month on the Expiring Leases graph.

Input Controls

You'll use the Input Controls to filter the report and set the Undecided Lease Alert. The input controls that appear are based on how you opened the Expiring Leases Report.

Status Filter

The possible statuses the leases can have are:

  • Given Notice
  • Renewed
  • Undecided
  • Month to Month (only available on report if opened from the Renewals report and the community/unit type has month to month leases)

By default, the report will display information for all statuses, but you can use this filter to only display the information for a particular status.

Undecided Lease Alert

Use this alert to highlight undecided leases that expire within a certain number of days.

By default, the notation for your undecided leases will be highlighted if they expire in 15 days or less.

This only affects the Undecided leases information on the report.

Type the number of days in the Undecided Lease Alert text box and click OK. In the Undecided leases section, the Notation column will be highlighted in yellow for any leases that expire in that number of days or less.

Group Leases By

By default, the leases are grouped by status, then community, but you can group them by community, then status, if you want to. You would do this if you have opened the report and then refreshed it and selected to view multiple communities. By default, the information is only for the community for the renewals number you selected on the Renewals report.

New Lease Signed

This allows you to filter the report to only show expiring leases which have a new lease signed or to only show expiring leases which do not have a new lease signed.

This filter is only available on the report when opened from the Renewals report.

Notations

The report contains a Notations column that is used to give you additional information about the expiring leases depending on the status of the lease.

For Undecided Leases, the notation will display the number of days until the lease expires. The notation will be highlighted in yellow if the lease expires in the number of days specified on the Undecided Lease Alert or less. For expired leases, the notation will display the number of days since the lease expired highlighted in red.

For leases with Given Notice and Moved Out status, the notation will display the new lease information if a new lease has been signed. If no new lease has been signed, the notation will be highlighted in orange.

For leases that have been renewed, the notation will display the lease term and rent amount for the renewal.

Tips

Prompts

The Expiring Leases report shows information based on the report it was opened from. However, you can click the Refresh button and then use the Prompts screen to view different information on the report without returning to the originating report.

If you opened the Expiring Leases report from the Renewals report, you can use the Prompts screen to change the dates, reporting group (hierarchy), community, and unit type.

If you opened the Expiring Leases report from the Portfolio Trends Dashboard, you can use the Prompts screen to change the reporting group (hierarchy), attribute, attribute value, community, and dates.

If you are changing a date, enter it in the format shown on the Prompts screen.

Navigation Map

You can use the Navigation Map to easily move to different sections of the report.
