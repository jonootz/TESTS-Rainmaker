KPI Report

The KPI Report can be accessed from the Reporting tab in the Web UI.

This report is typically used for performance tracking by regional managers, executives, and asset managers. It may be viewed as often as weekly. It gives you the ability to track multiple key performance indicators such as occupancy and rent growth over a specified period of time for a specific community or group of communities.

Viewing the Report

When you open the report, you'll be prompted to select a hierarchy. You can also select to view the report for a specific community or multiple communities.

To open the report:

  1. Select the KPI report on the Reporting tab in the LRO Web UI.

    The report will open in the Business Objects portal with the Prompts dialog shown.

  2. Under Prompts Summary, the Select Hierarchy prompt will be selected.

    If you don't see any hierarchies in the list, click Refresh Values.

    Select the hierarchy you want to use and then click the Add arrow button.

  3. If you want to view the report for a specific community or selection of communities, click the Select values for Community (ID) prompt.

    Selecting a community is not required. All communities that are included in the selected hierarchy will be included on the report if you don't select a community here.

    Click Refresh Values to see the list of communities.

  4. Select the community or communities that you want to view on the report and then click the Add arrow button.

  5. Click OK to view the report.

There are two views of the report, Overview and Detail, plus a Definitions page. You can switch between them using the tabs at the bottom of the screen.

Overview

The Overview shows you a graphical representation of the key performance indicators.

By default, the Overview page shows you a graph with average effective rent for leases in place, average effective rent for renewal leases, average effective rent for new leases, and occupancy. You can add average effective rent for expiring leases to the graph if you want to. The average effective rent for leases in place and occupancy will always show on the graph, but you can show/hide the other average effective rent information.

This is useful for comparing average effective rents and viewing occupancy trends.

The information is broken out Monthly by default, but you can view it broken out Daily, Weekly, or Quarterly, as well. If you select to view the information broken out Daily, there will be a large amount of data on the report so you may want to filter the report down to a specific community and/or unit type before choosing to view the information broken out Daily.

There is a page for each attribute value for the hierarchy level you are using to view the report.

You can use the Input Controls to filter the report to show the information exactly how you want it.

Detail View

The Detail view shows you the average effective rent and occupancy information in a table format.

You can view the information at the hierarchy level or at the hierarchy and community level or hierarchy, community & unit type level. You can also filter down to different levels in the hierarchy used to view the report.

You can select to view the information for a specified date range. You can also specify if you want to include amenities or month to month leases on the report.

You can choose to view the information for a particular community or unit type, as well.

Definitions
