Pricing Call Dashboard (BI Reporting)

The Pricing Call Dashboard can be run from the Reporting tab in the Web UI.

This dashboard is used by the person who is facilitating the pricing call. It summarizes the key screens you would use to facilitate or prepare for a pricing call (New Pricing, Pricing Trend Report, Availability, Competitor Summary, Lease Audit), which allows you to just look in one place instead of on several screens.

Viewing the Dashboard

This dashboard can be viewed at the community level and then at the unit type level.

Community Level View

Unit Type Level View

To view the pricing call dashboard for a community:

  1. Select Pricing Call Dashboard on the Reporting tab in the Web UI.

  2. In the Select Community drop-list, select the community you want to view information for.

To view the pricing call dashboard for a unit type:

  1. Select to view the pricing call dashboard for a community.

  2. In the Select Unit Type drop-list, select the unit type you want to view information for.

Tips

Sorting

You can sort the data in each section by clicking on a column heading.

Different Ways to View Information

On the Community View, the Current Availability and Exposure information and the Leasing and Renewal Activity information can be viewed as a chart or as a table.

On the Unit Type View, the Leasing Activity information can be viewed as a table or a chart.

Rent Comparison Graph Slider

On the Unit Type View, there is a slider at the bottom of the Rent Comparison graph that you can use to focus on sections of the graph. A miniature version of the graph appears above the slider so you can see which portion of the graph you are focused on.

Slider Showing Full Graph

Slider Focused on a Section of the Graph

Printing the Dashboard

To print the dashboard:

Click the Print icon located in the top right corner of the screen.
