Business Intelligence Reporting Admin and Setup: Properties

On the Reporting Admin Properties screen, you can view information about the properties that can be included in the reports. The property information comes over from the database as part of the nightly processes.

Properties are assigned to users under the Admin tab.

In the Web UI, go to Reporting Admin>Property.

You can click the link in the PMS Property ID field to see the details for a property.

Assigning Property Attributes

You'll need to set up attributes on the Property Attributes screen and then assign them to properties. Learn about setting up property attributes

Prior to release 14.11, you could set up a reporting hierarchy from the Admin tab under Communities>Hierarchy and if you had one set up there, the property attributes assigned there would initially be assigned here.

To assign attributes or change the attributes assigned to a property:

  1. In the Web UI, go to Reporting Admin>Property.

  2. Click the PMS Property ID link for the property.

  3. On the Property Details screen, click Edit.

  4. On the Edit Property Attribute dialog, select the value for any attributes that you want to assign to this property.

  5. Click OK.

Troubleshooting

I assigned an attribute value to a property, but that property is not included when I view my reports.

There are a couple of reasons that this could occur:

  • The report may be filtered so that the information is not being included. Be sure that you have selected a reporting group that includes this attribute (if applicable) and that your other filters would not exclude this property.
