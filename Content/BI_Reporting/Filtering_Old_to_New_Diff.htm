Filtering the Old to New Differential Report (BI Reporting)

You can filter the report by Community, Unit Type, Lease Type, Lease Terms, and Lease Start Date. The filters are the same on each view of the report with the exception of the Community filter. On the Overview, you cannot select multiple communities.

Community Filter

Overview

In the Overview, the Community filter allows you to run the report for all communities or a specific community.

By default, the report includes all communities that you have access to.

To filter the report for a specific community, select the community from the drop list.

Detail View

In the Detail view, the Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to.
