Daily Pulse Report (BI Reporting)

The Daily Pulse report can be accessed from the Reporting tab in the Web UI.

This report gives you a feel for where communities stand today in regards to recommended rent, exposure, occupancy, and average leases signed over the last 60 days. You can also look at the rent and exposure for the selected report date as compared with the prior day. This report is typically used by regional managers and owners.

Viewing the Report

The report will open when you click the Daily Pulse link on the Reporting tab.

When you open the report, you'll be prompted to select a date. You should select today's date or an earlier date. You'll also need to select a hierarchy. If desired, you can select to view the report for a specific community or multiple communities.

To open the report:

  1. Select the Daily Pulse report on the Reporting tab in the LRO Web UI.

    The report will open in the Business Objects portal with the Prompts dialog shown.

  2. Under Prompts Summary, the Select Date prompt will be selected.

    Select the date you want to run the report for. This should be today's date or an earlier date.

  3. Click the Select Hierarchy prompt.

    If you don't see any hierarchies in the list, click Refresh Values.

    Select the hierarchy you want to use and then click the Add arrow button.

  4. If you want to view the report for a specific community or selection of communities, click the Enter Community (ID) prompt.

    Selecting a community is not required. All communities that are included in the selected hierarchy will be included on the report if you don't select a community here.
    Click Refresh Values to see the list of communities, if necessary.

    Select the community or communities that you want to view on the report and then click the Add arrow button.

  5. Click OK to view the report.

There are two views of the report, Overview and Detail, plus a Definitions page. You can switch between them using the tabs at the bottom of the screen.

Overview

The Overview gives you a graphical representation of your daily average recommended rent for the past 30 days from the date you selected when you opened the report.

You can click a point on the graph to see key facts about your occupancy, exposure, effective rent, and recommended rent for that particular date.

The information that appears in the Overview is influenced by the input controls (filters).

Detail View

The Detail View gives you recommended rent comparisons at the community and unit type levels. You can also view Occupancy, Exposure, and Average Effective Rent information.

You can use the Input Controls to view the report for a specific community or group of communities. You can also view the report at different hierarchy levels using the Level to Display input control.

By default, the information is displayed expanded by community. You can roll up the information by community, if you don't want to see the details.

To do this:

  1. Select Outline in the toolbar if it is not already selected.

  2. Click on the Community Name of the community you want to roll up or expand.

  3. Click the arrow next to the community name to hide or show the details for that community.

Tips

Available Units

You can click on an exposure count to get details about the units that make up that count.

The information will open in a new report.

The Daily Pulse 60 Plus Available Units report shows information for the community or community and unit type for the count you selected. You can click the Refresh button and then use the Prompts dialog to view the Available Units for different communities, dates, and unit types.

There is also a Definitions tab to help you understand the information on the report.

New Lease Details

You can click on the 60 Day Avg Eff Rent for a community or community and unit type to get details about the new leases that make up that amount.

The information will open in a new report.

The Daily Pulse 60 Plus New Lease Details report shows information for the community or community and unit type for the average effective rent you selected. You can click the Refresh button and then use the Prompts dialog to view the New Lease Details for different hierarchies (reporting groups), communities, contract dates, and unit types.

There is also a Definitions tab to help you understand the information on the report.

Definitions

The last tab is a definitions page that will help you understand the information contained in the report.
