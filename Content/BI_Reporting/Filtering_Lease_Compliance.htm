Filtering the Lease Compliance Report (BI Reporting)

Filtering the Overview

You can use the Input Controls to filter the report and control how the information is displayed.

Community Name Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to that use an attribute in the selected hierarchy.

Level to Display Filter

By default, the report displays information for Level 1. For example, you may have a reporting group set up with Corporate as Level 1. The Corporate attribute has one value called Corporate and it is applied to all properties. This allows you to roll up information at the corporate level and then use the Level to Display filter to display the information broken out by other attributes. For example, if Level 4 is State, if you select to display Level 4, the report will be shown with the information broken out by state.

The Level to Display filter is useful when no other filters are selected or if you have selected multiple communities. It is not very useful if you have selected one filter item. For example, if you have selected one community in the community filter, the report will only show information for the selected community at every level you select.

When you view the report by level, each value for the level is shown on its own page. For example, if you selected Level 4 which is State, each page displays the report by a different state. You can see the attribute value in the filter box next to Hierarchy and Level Value.

Use the Level to Display drop-list to select the level you want to display.

Narrow Rent Comparison By

You can click on a bar in the Lease Count & Percentage at Compliance Rate graph to change the info in the Average CPR Effective Rent and Recommended Rent Comparison graph to only display for the selected month.

You can clear the selection using the Clear Filters button under Narrow Rent Comparison By in the Input Controls.

Filtering the Detail View

You'll use Input Controls and drill down functionality to filter the Detail view of the Lease Compliance report.

Input Controls

Community Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to that use an attribute in the selected hierarchy.

Level of Detail to Display

Learn how to view the report at different hierarchy levels

Show Compliant Leases As

The compliant leases will be shown as a percentage of all leases signed for that month. This filter allows you to display the actual number of leases that were compliant instead, if you want to.

The Compliance Alert is based on the percentage. Months where the compliant rate was below the specified percentage will still be highlighted if the compliant rate is shown as an amount.

Show Non Compliance Variance As

The non compliance variance is the difference between the actual effective rent and the recommended rent for the non-compliant leases. By default, this is shown as a percentage, but you can show it as a dollar amount, if you want to.

Compliance Alert

This is used to highlight months with a compliant rate that falls at or below a specified percentage.

Type the percentage in the Compliance Alert text box and click OK. Any months that have a compliant rate that falls at or below the specified percentage will be highlighted in yellow.

Display Leases

By default, the lease compliance information is broken out monthly to show the current month to date and the past 12 months. You can display this information quarterly instead, if you want to. Displaying the information quarterly will show the current quarter to date and the past 4 quarters.

Drill-Down Capability

You can drill down to levels in the hierarchy by clicking on the hierarchy attribute name. You can drill up by clicking on the arrow icon. You can hover over the name or the icon to see which level you will be drilling down or up to.
