Daily Pulse 60 Plus Available Units Report

The Daily Pulse 60 Plus Available Units report is opened from the Detail View of the Daily Pulse report. It gives you a way to get the details behind the 60 Plus Day Exposure Counts on the Daily Pulse report.

Opening the Report

To open the report, on the Detail View of the Daily Pulse report, click any of the numbers for the 60 Plus Day Exposure Count or 60 Plus Day Exposure Count Prior Day for the desired community or community and unit type to view the available units that make up those counts.

Input Controls

You'll use the Input Controls to filter the report.

Community Filter

This allows you to filter the report by community. When you open the report from the Daily Pulse report, it will only contain information for the community whose value you selected. The input control will only contain this community and is not useful in this case.

However, you can use the refresh button to view the report for different communities, dates, and unit types. This filter will be useful if you have refreshed the report and selected options that cause multiple communities to appear on the report.

Learn about refreshing the report

Unit Type

This allows you to filter the report by unit type. When you open the report from the Daily Pulse report, it will only contain information for the community whose value you selected. The input control will contain the unit types for the selected community or if you selected a particular unit type, it will only contain that unit type.

You can use the refresh button to view the report for different communities, dates, and unit types. This filter will be useful if you have refreshed the report and selected options that cause multiple unit types to appear on the report or if you opened the report by selecting the exposure count for a community with multiple unit types.

Learn about refreshing the report

Status Filter

The possible statuses the units can have are:

  • On Notice
  • Vacant Ready
  • Vacant Unavailable

By default, the report will display information for all statuses, but you can use this filter to only display the information for a particular status.

The statuses that appear in the Input Control are those that are applicable to the units displayed on the report. For example, if you only have one unit on the report and its status is "Vacant Ready", then only "Select (All)" and "Vacant Ready" will appear in the Input Control.

Availability Filter

By default, the report shows units that are available in the next 365 days. You can show units that are available in a shorter time frame if you want to. The filter options are:

  • Now
  • Next 30 Days
  • Next 60 Days
  • Next 90 Days
  • Next 120 Days
  • Next 365 Days

Report Type

The report can be viewed in Detail or Summary view. The Detail view shows the pricing information for all lease terms for each unit. The Summary view shows the pricing information for the best lease term for each unit.

The report is displayed in Summary view, by default.

Group By

By default, the units are grouped by unit type. You can group them by unit status or availability, if you want to.

Show Square Footage

You can use this filter to include square footage on the report. If you select Yes, the square footage information will be included in the Amenities column.

Tips

Prompts

The Daily Pulse 60 Plus Available Units page shows information for the community or community and unit type for the count you selected. You can click the Refresh button and then use the Prompts dialog to view the Available Units for different communities, dates, and unit types.

Definitions

There is also a Definitions tab to help you understand the information on the report.
