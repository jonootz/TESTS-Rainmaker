Business Intelligence Reporting Admin and Setup: Property Groups

On the Reporting Admin Property Groups screen, you can view information about the property groups, including which properties make up a particular group. This information comes over from Full Groups and Property Groups from the user setup under the Admin tab as part of the nightly processes.

Property groups are assigned to users in the user setup under the Admin tab.

To view your property groups:

  1. In the Web UI, go to Reporting Admin>Property Group.

  2. For any property group, click View Details to see which properties are included in that property group.
