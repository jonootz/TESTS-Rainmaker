Filtering the Renewals Report (BI Reporting)

The Renewals Report has two views, Overview and Detail. You can filter both of these views using Input Controls and by drilling down on the report to different hierarchy levels.

Filtering the Overview

The Overview page can be filtered by drilling down on the graph labels. You can use the Filter Bar at the top of the report to change the drill selections. You can also use the filter bar to see the levels you have drilled down through to get to the current one.

To drill down on the Overview page:

Click on the hierarchy label to drill down to the next level in the hierarchy. When you hover over the label, you can see which level you will be drilling down to.

Both graphs will change to show details from the next level in the hierarchy. You can see which levels you have drilled down to at the top of the report and in the Filter Bar above the report.

To drill back up, click the Level on either graph. This will change both graphs to show details from one level higher than the current level.

By default, the information shown on both graphs is for the current month, but you can use the Input Controls to view it for the next month, two months out, or three months out, as well.

The Variance graph will not be displayed if you choose to view the information for three months out because Up-Down information is not available three months out.

Tip

When you open the report, you will be prompted to select a hierarchy. If you want to view the report using a different hierarchy, you can close and reopen it or select the refresh button.

Hierarchies are also known as Reporting Groups and are set up under the Reporting Admin tab.

Filtering the Detail View

You'll use Input Controls and drill down functionality to filter the Detail view of the Renewals report.

Input Controls

Community Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to that use an attribute in the selected hierarchy.

Level of Detail to Display

You can control the amount of information you want to display on the report. You can view it at three levels of detail:

  • Hierarchy Only - information is shown at the hierarchy attribute level
  • Hierarchy & Community - information is shown at the hierarchy attribute level broken down by community
  • Hierarchy, Community & Unit Type - information is shown at the hierarchy attribute level broken down by community and then by unit type

For example, let's say you're viewing the report at Level 3 which is Region. The report will be broken out by Region if you view it as Hierarchy Only. The report will be broken out by Region and then by Community within each region if you view it as Hierarchy & Community. The report will be broken out by Region, then by Community within each region, then by Unit Type within each community if you view it as Hierarchy, Community & Unit Type.

Drill-Down Capability

You can drill down to levels in the hierarchy by clicking on the hierarchy attribute name. You can drill up by clicking on the arrow icon. You can hover over the name or the icon to see which level you will be drilling down or up to.
