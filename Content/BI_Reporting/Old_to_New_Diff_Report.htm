Old to New Differential Report (BI Reporting)

The Old to New Differential Report can be accessed from the Reporting tab in the Web UI.

This report is useful for comparing metrics at a detailed level for your old leases vs. your new leases. The Overview helps you visualize your leasing trends from lease to lease.

Month to Month leases are not included on this report.

What questions can the Old to New Differential Report answer?

Viewing the Report

When you open the report, you'll be prompted to enter the date you want to run the report for. This would typically be today's date.

Learn about filtering the Old to New Differential Report

To run the Old to New Differential Report:

  1. Select the Old to New Differential report on the Reporting tab in the LRO Web UI.

  1. On the Report Date prompt, select the date you want to run the report for. Most likely, this will be today's date.

  2. Click OK to run the report.

There are two views of the report, Overview and Detail. You can switch between the two views using the tabs at the bottom of the screen.

Overview

The Overview summarizes the information on the Old to New Differential report using a table and graphs.

Average Rent Variance and Lease Count by Month

You get a summary table and a graph that show you the Average Rent Variance and Lease Count by Month based on your input control selections. You can view this information for all communities, unit types, lease types, lease terms, and lease start dates or you can filter the report by any or all of those options.

Average Rent Variance: Average difference between the rent amount for the new leases for the month vs. the previous leases

Number of Leases: Number of new leases for the month

% Change: Percentage difference between the rent amount of the new leases for the month vs. the previous leases

Average Rent Variance by Lease Term

There is another graph that shows you Average Rent Variance by Lease Term. You can also see the number of new leases for each lease term. This is also influenced by the input controls (filters).

Average Rent Variance: Average difference between the rent amount for the new leases vs. the previous leases broken down by lease term

Number of Leases: Number of new leases for each lease term

New Lease Rent vs Old Lease Rent

There is a pie chart that allows you to compare how many new leases were signed at a higher, lower, or same rent as the previous lease. This is influenced by the input controls (filters).

Detail View

The Detail View of the Old to New Differential report allows you to compare new lease information to the previous lease information for a particular unit summarized by unit type.

The information is grouped by community, then unit type.

Community Name (ID): Name and ID of the community

Unit Type: unit type

Unit: Unit ID

Old Lease

Resident Name: Resident on the previous lease for the unit

Lease End Date: Date the previous lease for the unit ended

Lease Term: Lease term for the previous lease for the unit (Also shown as an average per unit type and per community)

Rent: Previous rent amount for the unit (Also shown as an average per unit type and per community)

New Lease

Resident Name: Resident on the new lease for the unit

Lease Start Date: Date the new lease for the unit started

Application Date: Date the resident applied for the lease for the unit

Lease Term: Lease term for the new lease for the unit (Also shown as an average per unit type and per community)

Rent: New rent amount for the unit (Also shown as an average per unit type and per community)

Old to New

Diff: Difference between the new rent and the previous rent for the unit (Also shown as an average per unit type and per community)

% Change: Percentage difference between the new rent and the previous rent for the unit (Also shown as an average per unit type and per community)

The unit ID and lease information come from your PMS system.

Tip

You can show and hide the details on the Detail View.

To do this:

  1. Select Outline in the toolbar if it is not already selected.

  1. Click on the Community Name of the community you want to expand or hide.

  2. Click the arrow next to the community name to show or hide the details for that community.
