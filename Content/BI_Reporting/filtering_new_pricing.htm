Filtering the New Pricing Report (BI Reporting)

The New Pricing Report has two views (Overview and Detail). Each view of the report has its own Input Controls (filters). For example, if you select to view the Overview for a particular community and then switch to the Detail view, you will have to select that particular community again for the Detail view, if that is the only community you want to see.

Filtering the Overview

The Overview can be filtered by Community, Status, Availability, Pipeline, and Unit Type.

Community Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

Click OK in the Community filter area to apply this filter to the report.
