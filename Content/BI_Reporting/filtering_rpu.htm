Filtering the Revenue Per Unit (RPU) Report (BI Reporting)

You can use the Input Controls to filter the report. In addition to filtering, you can specify if you want the rent amounts and related calculations on the report to include amenities or not. You can also specify if you want to display RPU on the report as either a bar or a line.

Chart RPU As Filter

By default, RPU is shown as a bar, but you can use this filter to show it as a line instead.

Viewing RPU as a bar allows you to quickly compare the RPU with the average rent for your occupied units since average rent is represented by a bar.

Viewing RPU as a line allows you to quickly see how your RPU is trending with your occupancy which is represented by a line.

Select Month

By default, the report shows RPU for the current month and the past 16 months. You can also view the report for only one month or for a selection of months. For example, you may want to exclude months from the report that don't have data, in the case of a lease up, or to only view the report for a particular set of months, such as a particular quarter.

To select multiple months, hold down the Ctrl key while you select each month.
If you want to select a group of sequential months, select the first month, then hold down the Shift key while you select the last month. The first and last months, plus all the months in between, will be selected.

Click OK in the Select Month filter area to apply this filter to the report.

Drill Down

In addition to the Input Controls mentioned previously, you can filter the Drill view by drilling down to the different levels on the report using the headings. A page of the report is created for each attribute when you drill down.

You can click a heading to drill down a level or you can click the icon next to the heading to go up a level.

When you mouse over a heading or an icon, you'll see what level you will be drilling down or up to.

Each heading is a value of a particular attribute. The attributes (levels) come from the Reporting Group (hierarchy) set up under the Reporting Admin tab. The headings (attribute values) are also set up under the Reporting Admin tab.

Tips

You can see the levels you have drilled down to in the Filter Bar. You can make different selections for a level using the Filter Bar if you want to.

If you have selected a level that has quite a few attributes, you may want to use the Navigation Map to easily navigate to each page of the report, instead of paging through the report.

You can take a snapshot of the RPU report, if you want to. This is great for comparing information. For example, you might drill down to a particular region and take a snapshot. Then drill down to a different region and take a snapshot. Then you can easily compare the snapshots of the two regions.
