Filtering the Competitor Age Report (BI Reporting)

You can use the Input Controls to filter the report by Community, Competitor, Status, and Days Since Updated. You can also display rent history on the report, if you want to.

Community Filter

By default, the report includes all communities that you have access to.

Competitor (Code) Filter

This allows you to filter the report to only see information for a particular competitor or selection of competitors.

To select multiple competitors, hold down the Ctrl key while you select each competitor.
If you want to select a group of sequential competitors, select the first competitors, then hold down the Shift key while you select the last competitor. The first and last competitors, plus all the competitors in between, will be selected.

Status Filter

This allows you to filter the report to see all competitors or only competitors that are stale or only competitors that are not stale.

Days Since Updated Filter

You can also filter the report based on the days since the competitor rent has been changed. For example, you would enter 20 to see any competitors whose rents haven't changed in 20 or more days. This works off the Rent Change Aging.

Show Rent History

You can use this filter to see competitor rent history going back 12 weeks from the report date.
