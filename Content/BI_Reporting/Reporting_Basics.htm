Business Intelligence Reporting Basics

Some basic functionality that you will use on any of the Business Intelligence reports includes printing reports and exporting them to Excel.

Most of these instructions are for the reports, not the dashboards.

Printing a Report

  1. Click the Print button in the toolbar.

    This will create a PDF that you can use to print the report.

  2. Click Open on the message asking if you want to open or save the PDF file. (This message appears at the bottom of the screen.)

  3. Click Print on the Adobe Reader toolbar to print the report.

Finding Information on a Report

This is useful for finding information in Detail views of reports.

  1. Click the Find button in the toolbar.

    The Search Bar text box will appear at the bottom of the report.

  2. Enter the information that you want to find.
  3. Press Enter or the Find button next to the Search Bar text box.

  4. If no matches are found, you'll see a message next to the Search Bar text box that says "No match on current page" and your search information will appear in red.

    If matches are found, you'll see a message next to the Search Bar text box that says "Found matches on current page" with Find Next and Find Previous options. The first matching text item will be highlighted. Use the Find Next or Find Previous options to highlight the next or previous matching item.

    This only searches the current page. If you have multiple pages, you will have to repeat the search on each page that you want to include.

Depending on your browser, you may see this symbol next to the found information. If so, just ignore it.

Freezing Headers

You can freeze the row header and column header on a report, if necessary. This allows you to scroll though the information on a report without losing the header information.

Freezing headers is not "sticky". You have to freeze the headers each time you view the report, if necessary.

To freeze both the row header and column header:
  1. Click anywhere in the report.

  2. Click the Freeze button in the toolbar above the report.

To freeze only the row header:

  1. Click anywhere in the report.

  2. Click the down arrow next to the Freeze button in the toolbar above the report.

  3. Check Freeze Header Rows.

To freeze only the column header:

  1. Click anywhere in the report.

  2. Click the down arrow next to the Freeze button in the toolbar above the report.

  3. Check Freeze Header Columns.

To undo freezing the headers:

  1. Click anywhere in the report.

  2. Click the down arrow next to the Freeze button in the toolbar above the report.

  3. Uncheck the freeze header options.

    The menu will close after you uncheck one option so you'll need to repeat the steps if you need to uncheck more than one option.

Exporting a Report to a CSV File

  1. Select Export Data to CSV.

  2. On the Save as CSV Options dialog, change the options for the CSV file, if necessary.

    Click OK.

  3. Click Save on the message asking if you want to open or save the CSV file. (This message appears at the bottom of the screen.)

Exporting a Report to PDF

  1. Select PDF.

  2. Click Save on the message asking if you want to open or save the PDF file.

Exporting a Report to Excel

  1. Select Excel. (Choose Excel 2007 if you are using that version of Excel.)

  2. Click Save on the message asking if you want to open or save the Excel file.

The column and row headings may appear as hyperlinks when you export the report to Excel. However, they are not actually hyperlinks and clicking on them won't do anything.

Taking Snapshots

You can take a snapshot of any "drillable" report, if you want to. This is great for comparing information. For example, if you have region set up as an attribute in your hierarchy, you might drill down to a particular region and take a snapshot. Then drill down to a different region and take a snapshot. Then you can easily compare the snapshots of the two regions.

When you take a snapshot, the page is saved with the drill down options to a new tab. You can print this tab or export it to Excel.

Snapshots are only available while you are viewing the report. They will not be there after you close the report and open it again.

To take a snapshot of a "drillable" page:

  1. Click the Arrow next to the Drill button in the toolbar.

  2. Select Snapshot.

The page will be saved in a new tab with any drill down selections that you have made.

In order to see the drill down selections that were made on the report when the snapshot was taken, click the Filter Bar button in the toolbar.

You cannot take a snapshot of dashboards. For non-drillable reports, you can click Drill and then select Snapshot to save a snapshot of the page. However, you will need to click Drill again to turn it off so you can use the filters for the report.

Information About The Report

In the top right area of most reports, you can see who ran the report, when it was run, and how up-to-date the data on the report is. The data should be loaded each night as part of the nightly processes.

On the dashboards, you'll see when the data was loaded and when the dashboard was refreshed.

Business Objects Portal Session

If the Business Objects portal session times out, you will be prompted to log in directly to the Business Objects portal. However, you will get an error if you try to do this.

You should close the Business Objects portal window and reopen the report or dashboard from the Reporting tab. You may need to log into the Web UI again if that session has timed out as well.
