Filtering the Rent Trend Report (BI Reporting)

You can filter both views of the report using the input controls and by drilling down.

Input Controls

Amenities Filter

Community Filter

Unit Type Filter

Select Specific Quarters

Click OK in the Select Specific Quarters area to apply this filter to the report.

Drilling Down

In addition to the Input Controls mentioned previously, you can filter each view by drilling down to the different levels on the report using the headings. A page of the report is created for each attribute when you drill down.

You can click a heading to drill down a level or you can click the icon next to the heading to go up a level.

Overview

Detail View

When you mouse over a heading or an icon, you'll see what level you will be drilling down or up to.

Overview

Detail View

Each heading is a value of a particular attribute. The attributes (levels) come from the Reporting Group (hierarchy) set up under the Reporting Admin tab. The headings (attribute values) are also set up under the Reporting Admin tab.

Tips

You can see the levels you have drilled down to in the Filter Bar. You can make different selections for a level using the Filter Bar if you want to.

You can take a snapshot of either view of the report, if you want to. This is great for comparing information. For example, you might drill down to a particular region and take a snapshot. Then drill down to a different region and take a snapshot. Then you can easily compare the snapshots of the two regions.
