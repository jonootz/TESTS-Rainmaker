Portfolio Trends Dashboard (BI Reporting)

The Portfolio Trends Dashboard can be run from the Reporting tab in the Web UI.

This dashboard is useful for anyone that wants to gauge performance at a community or group of communities. It will most likely be used by someone in the corporate office such as a VP, Regional Manager, Pricing Analyst, or Owner.

Much of the information you'll find on the Portfolio Trends Dashboard is also displayed in the Weekly Report. The Weekly Report provides a snapshot in time for a community, which provides information on where the community stands today and what has happened over the last 60 days. The Portfolio Trends Dashboard helps illustrate this information through charts and graphs. The Occupancy and Rent Trends section shows a 90 day trend. You can also view the information on the dashboard in a table view by clicking on the table icon in the upper right corner of each section.

The Expiring Leases section provides additional information that is not on the Weekly Report. It gives you insight into the renewal retention for residents with expiring leases in the current month and the next month.

Viewing the Dashboard

To view the Portfolio Trends Dashboard:

  1. Select Portfolio Trends Dashboard on the Reporting tab in the Web UI.

  1. Select the hierarchy (reporting group) that you want to use to view the dashboard.

  2. Select the attribute you want to use to view the dashboard.

  3. Select the attribute value you want to use to view the dashboard.

    After you select an attribute value, data will populate in the dashboard.

  4. If applicable, you can filter the dashboard further by selecting a community.

    To see the dashboard for all communities, select Corporate as the Attribute and Corporate as the Attribute Value. Don't select a Community. This is a common hierarchy structure that you probably have set up. All communities that have the Corporate attribute value assigned to them will be included. This is set up under the Reporting Admin tab in the Web UI.

Tips

Hierarchies (Reporting Groups)

The Hierarchies, Attributes, and Attribute Values come from the Reporting Groups that are set up under the Reporting Admin tab.

You may want to set up a reporting group specifically for the Portfolio Trends Dashboard. Reporting Groups determine which information you see on a report or dashboard and how you can filter that information. For example, you may want to be able to view the information by Region or City. Your existing hierarchy or hierarchies may already be set up with the selections you need for this dashboard, but just know that you can set up a hierarchy to use specifically for this dashboard if you want to. If you do set up a reporting group specifically for this dashboard, you may want to include "PTD" or "Portfolio Trend" or something in the name so you can easily identify it in the hierarchy list.

Only information for communities that are assigned the selected attribute value will be included in the dashboard. Attribute values are assigned in the Property section under the Reporting Admin tab in the Web UI.

Table View

You can click the Table icon in the top right corner of a section of the dashboard to view the information that makes up the graph.

Then, you can select the Graph icon in the top right corner of the Table view to change it back to a graph.

Printing the Dashboard

To print the dashboard:

Click the Print icon located in the top right corner of the screen.

Understanding the Dashboard

Occupancy and Rent Trends

This section gives you an overview of what is happening with your occupancy and leases over the last 90 days. You'll be able to see the following information in this section. You can hide/show this information using the checkboxes at the top of the graph. You can also select whether or not you want to include amenities in the rent amounts.

Occupancy: Occupied Units/Total Units

All Leases: Average rent for all leases that are currently in place

Expiring: Average rent for all leases that have expired in the last 60 days

Renewing: Average rent for all leases that have renewed in the last 60 days

Move-ins: Average rent for all new leases with a move-in date in the last 60 days

Asking Rent: Average recommended rents. This allows you to see your rent trend. (The asking rent is always shown without amenities.)

The dates at the bottom of the graph are the week start dates.

Exposure

This section shows you how many apartments are vacant available or notice available in a given time period.

Vacant: Units that are vacant available with an available date less than or equal to today's date

30 Day: Units that are vacant/notice available in the next 30 days

60 Day: Units that are vacant/notice available in the next 60 days

Total: Total of the vacant/notice available units from today until 60+ days from now

Leasing

This section will show you how many guest cards/visits have occurred in the last 7 days and how many leases have been signed in the last 7 days.

Renewal Percentage

This section shows you how many leases expired in the prior calendar month and how many of those residents renewed their lease.

Expiring Leases

This section displays information about leases that are expiring in the current calendar month or the next calendar month. You can use this section to see how many residents with expiring leases in the current month and next month have renewed, given notice, or are undecided at this point.
