Filtering the Availability & Pricing Report (BI Reporting)

The Availability & Pricing Report has two views, Overview and Detail. You can filter both of these views using Input Controls and by drilling down on the report to different hierarchy levels.

Filtering the Overview

The Overview page can be filtered by drilling down on the graph labels. You can use the Filter Bar at the top of the report to change the drill selections. You can also use the filter bar to see the levels you have drilled down through to get to the current one.

To drill down on the Overview page:

Click on the hierarchy label on either graph to drill down to the next level in the hierarchy. When you hover over the label, you can see which level you will be drilling down to.

Both graphs will change to show details from the next level in the hierarchy. You can see which levels you have drilled down to at the top of the report and in the Filter Bar above the report.

To drill back up, click the Level on either graph. This will change both graphs to show details from one level higher than the current level.

You can also use the Input Controls to filter the information on the Overview.

Amenities

Community Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to that use an attribute in the selected hierarchy.

Levels 1 - 3 Filters

You can choose to display information on the graphs for a particular attribute value in a level. For example, if level 3 is State, you could choose to view the information for a particular state.

The filter for each level allows you to narrow the information down by attribute values. You can also see which values are included in each level.

A reporting group can contain up to 10 levels, but it is uncommon to have that many. You can view the graph at a levels 1-10 by drilling down, but you can only filter the report by the attribute values for levels 1 - 3.

Tip

When you open the report, you will be prompted to select a date and a hierarchy. If you want to view the report using a different date or hierarchy, you can close and reopen it or select the refresh button.

Hierarchies are also known as Reporting Groups and are set up under the Reporting Admin tab.

Filtering the Detail View

You'll use Input Controls and drill down functionality to filter the Detail view of the Availability & Pricing report.

Input Controls

Show Comparisons As

This allows you to show your occupancy and average rent from last week, last month, last year, and first of the year as percentage changes or as the actual measures for those time periods.

Amenities Filter

If you set this filter on the Detail view, it will persist to the Overview and vice versa.

Community Filter

The Community filter allows you to run the report for all communities, multiple communities, or a specific community.

By default, the report includes all communities that you have access to that use an attribute in the selected hierarchy.

Drill-Down Capability

You can drill down to levels in the hierarchy by clicking on the hierarchy attribute name. You can drill up by clicking on the arrow icon. You can hover over the name or the icon to see which level you will be drilling down or up to.
