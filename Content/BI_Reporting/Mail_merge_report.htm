Renewals Mail Merge Report (BI Reporting)

The Renewals Mail Merge report can be run from the Reporting tab in the LRO Web UI.

This report is used to create renewal letters. It allows you to export pricing for all expiring leases and renewal offers for the selected period of time. This report would typically be run whenever you are generating renewal offers, which is usually monthly.

Viewing the Report

When you open the report, you'll be given a prompt to make your filter selections and run the report.

To run the mail merge report:

  1. Select the Mail Merge report on the Reporting tab in the LRO Web UI.

    The report will open in the Business Objects portal with the Prompts dialog shown.

  2. Make sure the first prompt under Prompts Summary is selected (Enter value for Curr Lease Exp Date (Start)).

  3. Select or enter the start date.

  4. Select the second prompt under Prompts Summary (Enter value for Curr Lease Exp Date (End)).

  5. Select or enter the end date.

    If you accidentally enter a start date that is later than the end date, the report will be blank.

  6. Select the third prompt under Prompts Summary (Enter values for Community).

  7. Select the communities you want to include on the report.

    If you select All here, only All will be available as the Unit Type selection. If you want to filter this report by unit type, you should select specific communities here, instead of selecting All. If you want to filter by unit type and you want all communities on the report, just select all of them. You can select multiple communities using Shift-click or Ctrl-click.

  8. Select the Add button after you have made your selection. The list on the right will show the communities that are selected.

  9. Select the fourth prompt under Prompts Summary (Enter values for Unit Type).

  10. Select the unit types you want to include on the report. You can select multiple unit types using Shift-click or Ctrl-click.

    If you selected All for the Community value, only All will be available as the Unit Type selection. If you want to filter this report by unit type, you should select specific communities under Enter values for Community, instead of selecting All. If you want all communities on the report, just select all of them.

  11. Select the Add button after you have made your selection. The list on the right will show the unit types that are selected.

  12. Click OK to run the report.

Tips

If your report is very large, you'll use the page functionality instead of continuously scrolling down.

To rerun the report with different filter selections, you can click the Refresh button.

The filters for this report are all on the Prompt screen that appears when you run the report. This report does not use filters from the Input Controls or Filter Bar like some of the other reports do. Since these filters are not available, you may see a note about adding them. You do not need those filters for this report so you should not try to add them.
