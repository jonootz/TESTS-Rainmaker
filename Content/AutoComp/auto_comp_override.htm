Reviewing and Overriding Competitor Rents from Automated Comps

By default, Automated Comps will get comp rents two days before the DRRC is scheduled to run. This way you can review the rents and address any issues or make any overrides before the DRRC runs for the week.

You can also manually run Automated Comps by clicking Run Auto Comp on the Comp Rents screen. This runs Automated Comps for all competitors for that community who are set up to use Automated Comps.

You can change the number of days using parameter 320.

Reviewing Automated Comps Rents

To review Automated Comps rents, you'll go to LRO>Competitors>Comp Rents.

For competitors who use Automated Comps, you'll just need to go review the rents each week to make sure that new rents were retrieved and that they are ok. If there are any problems, you'll see exceptions at the top of the screen. Also, there are circle icons that represent each unit type for a competitor which is set up for Automated Comps. These circles should be green. If there is a problem with the rent for a unit type, the corresponding circle will be yellow-orange. If Automated Comps is turned on for the competitor, but not for a unit type, the corresponding circle will be grey.

You can update the contact info and shopping info (visits, leases, occupancy, shop method) at the top of the screen, if necessary. Simply, click on the information you want to update and change it. When you click out of the field, the information will be saved.

Overriding Automated Comps Rents

If the base rent is not correct or if the competitor is offering a concession, you can override the Automated Comps rent. To do this:

  1. Go to LRO>Competitors>Comp Rents.

  2. On the Comp Rents screen, select the competitor you want to update on the left.

  3. Set the toggle under the Comp Rent that you want to change to Override.

  4. Enter the Base Rent and Concession, if applicable.

    The Base Rent is the starting rent for a 12 month lease for that floorplan. The Concession is the total dollar value of the concession for a 12 month lease.

  5. Click the Green checkmark to save the rent.

Keep in Mind

  • Once you have overridden an Automated Comps rent, that rent will remain in place until the next time Automated Comps runs after the DRRC has run. This ensures your override is used in the next DRRC run.
  • The next time Automated Comps runs after the DRRC has run, it will get a new rent for this comp unit type and the toggle will be set back to Auto Comp.
  • If you manually change the toggle back to Auto Comp, then Automated Comps will get a new rent the next time it runs regardless of whether or not the DRRC has run.
