Automated Comps Statuses

You can see which competitors are set up to use Automated Comps on the Comp Setup screen.

These are the possible statuses that you will see for a competitor in the Auto Comp column.

Status Reason

Off

Automated Comps is not turned on for the community. All competitors will have the Off status. You will need to shop and enter comp rents for this competitor.

or

Automated Comps is turned on for the community. The competitor was entered as an Automated Comps competitor, but then Automated Comps was turned off for the competitor. You can turn Automated Comps on for this competitor. Otherwise, you will need to shop and manually enter comp rents for this competitor.

or

The competitor is a sister community. Sister communities cannot use Automated Comps.

N/A

This competitor was manually entered. Automated Comps is disabled for the competitor and there is no Automated Comps match.

You will need to shop and manually enter comp rents for the competitor.

This competitor was manually entered. Automated Comps is disabled for the competitor, but there is an Automated Comps match.

You can turn Automated Comps on for this competitor. Otherwise, you will need to shop and manually enter comp rents for this competitor.

Automated Comps is enabled for this competitor.
