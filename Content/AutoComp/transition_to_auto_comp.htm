Transitioning to Automated Comps

When you turn Automated Comps on for a community, the next step is to transition your existing competitors to use Automated Comps.

Automated Comps is turned on for a community using parameter 317.

Transitioning an Existing Competitor to Use Automated Comps

You can transition existing competitors with Automated Comps statuses of Match or Off to use Automated Comps.

When you transition an existing competitor to Automated Comps, it keeps the pricing history for that competitor.

Transitioning a Competitor with Match Status

After Automated Comps has been turned on for your community and the nightly processes have run, any existing competitors that Automated Comps has found a match for will show in the Auto Comp column on the Comp Setup screen.

To convert a matched competitor to use Automated Comps:

  1. On the Comp Setup screen, in the Auto Comp column, click .

    The Update Competitor screen will open with the competitor pinned on the map and listed to the left of the map.

    If this isn't the correct competitor, you can clear out the search text and search for another competitor.

  2. Select the competitor.

  3. Click the Add button.

  4. Next, you'll need to verify the unit type mappings and bed, bath, and square foot information for each unit type. Learn how to do this.

Transitioning a Competitor with Off Status

To turn Automated Comps back on for a competitor that used to be set up to use Automated Comps:

  1. On the Comp Setup screen, click the pencil icon in the Actions column for the competitor.
  2. Click Edit on the Competitor Information section.

  3. Change the Automated Comp Settings toggle to On.

  4. Click Save.

    If the comp unit types were set up for Automated Comps before, then you don't have to do anything else. Otherwise, you will need to set up the competitor unit types to use Automated Comps.

You cannot turn Automated Comps on for a competitor if it is a sister community or if Automated Comps is not turned on for the community.

Tips

If a competitor is showing the N/A status, but you can see it on the Add Competitor map, double-check the address you have. If it is different, edit the address of the competitor to match the one shown on the Add Competitors map. Then after the nightly processes run, it should be marked with the Match status and you can easily convert it to use Automated Comps. Another option is to turn Automated Comps on for the competitor (if the competitor has address information), then find and select the competitor on the map.
