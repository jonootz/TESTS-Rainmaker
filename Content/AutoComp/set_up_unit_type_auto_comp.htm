Setting Up a Competitor Unit Type to Use Automated Comps

After you have turned Automated Comps on for a competitor or set up a new competitor to use Automated Comps, you'll need to set up the comp unit types for that competitor to use Automated Comps.

To do this:

  1. Go to LRO>Competitors>Comp Setup.

  2. On the Comp Setup screen, click the pencil icon on the row for the competitor.

  3. Click on the unit type in the Unit Types section of the screen.

  4. Verify the unit type information and mapping.

  5. Set the Automated Comp Settings toggle to On, if necessary.

  6. Enter the bed, bath, and square foot information, if necessary.

  7. Click Save.
  8. Repeat steps 3 - 7 for each competitor unit type.

If Automated Comps is not able to find a rent for the unit type, you'll see an error message on the competitor unit type screen. Make sure the bed, bath, and square foot range actually matches a floorplan of the competitor. You will need to shop any unit types that Automated Comps cannot find rents for and manually enter the rent information.

It's possible that the information is set up correctly, but there are no units currently available for that unit type. In this case, Automated Comps will not be able to find a rent. You should just shop and manually enter the rents for the unit type until a unit of that unit type becomes available and Automated Comps can get a rent.
