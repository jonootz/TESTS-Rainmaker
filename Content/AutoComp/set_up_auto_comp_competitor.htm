Setting Up New Automated Comps Competitors

If you have Automated Comps turned on for your community, you can set up new competitors to use Automated Comps.

You can also transition existing competitors to use Automated Comps.

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click Add Competitor.

  3. Potential competitors and existing competitors will appear on the map and in a list on the left side of the screen.

    Competitors that are already set up will appear in the list with a green checkmark.

    Select each community that you want to use as a competitor. You should select an Automated Comps community. Automated Comps communities are marked with the dark blue pin. If you don't see a particular community on the map with a dark blue pin, then it is not eligible to use Automated Comps.

  4. Click the Add button.

  5. On the Comp Setup screen, click the pencil icon on the row for the competitor you just added.
  6. Verify the competitor information is correct. If not, click the Edit button for that section and change it. You can also enter information about why this community is a competitor and the competitor's utilities, amenities, and pet policies.
  7. Make sure the unit types are set up correctly for Automated Comps.
  8. Repeat steps 5-7 for each competitor that you added.
  9. Click Close.

After setting up your competitors, you'll need to enter competitor weights and positions.
