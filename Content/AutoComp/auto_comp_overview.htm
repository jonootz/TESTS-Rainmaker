Automated Comps Overview

Automated Comps makes it easy to enter your competitor rents each week. It automatically gets the rent information for the competitors you have set up to use Automated Comps. That means all you have to do is review the rent information instead of calling around or looking it up each week.

Automated Comps is a premium feature that is available for an additional fee. Contact your account representative for more information.
