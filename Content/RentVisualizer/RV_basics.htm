Rent Visualizer Basics

Viewing the Rent Visualizer

The Rent Visualizer is opened from the New Pricing screen. You can click the Base Rent to see the Rent Visualizer for that Unit Type and Lease Term Category.

The link is from the Base Rent and not the Effective Rent because the Effective Rent will not always match the Rent Visualizer information because the Rent Visualizer does not include concessions. Also, be aware that the Base Rent on New Pricing may not match what is in the Rent Visualizer because amenities and pricing offsets are included in New Pricing, but not in the Rent Visualizer.

The Rent Visualizer will open in a separate tab. This allows you to switch back and forth between the Rent Visualizer and the New Pricing screen if you want to.

The Rent Visualizer is divided into five components to help you understand the pricing process. Inputs and parameters that were used at each pricing stage are displayed in the panel on the right side of the screen when you select a point on the graph.

This gives you some insight into the pricing calculations and what affects them so you can make adjustments or just have a better general understanding of LRO's pricing process.

Filtering the Rent Visualizer

When you open the Rent Visualizer, you will see the rent information for the unit type and lease term category for the base rent you selected. You can filter the Rent Visualizer to see different information.

Pricing Date

You can select to view the pricing as of the date the pricing was run.

The available dates will start from the date you started using the Rent Visualizer functionality. If pricing wasn't run for a particular day or if the Rent Visualizer functionality was turned off (parameter 313), that date will not be in the list. Also, information for dates more than 3 months back may have been purged and no longer be available.

Pricing Week

You can select to view the pricing for a specific week. For any given Pricing Date, you can select pricing for a number of weeks into the future. The number of weeks is determined by parameter 1, Opt_Horizon, which is set to 16 by default.

LTC

You can select to view the pricing for a particular lease term.

The Lease Terms filter will change to display lease terms for the lease term category selected here.

Unit Category

You can select to view the pricing for a particular unit category.

The Unit Types filter will change to display unit types that fall within the selected unit category.

Unit Types

You can select any unit types that fall within the selected unit category.

You must select at least one unit type.

Lease Terms

You can select any lease terms that apply for the selected lease term category.

You must select at least one lease term.

Tips

The labels on the graph will help you identify the pricing path for a particular unit type and lease term. These labels are on the starting point (Reference Rent) and ending point (Further Post-Processing). This is helpful if you have filtered the report to show multiple unit types or multiple lease terms.

You can also mouse over a point on the graph to see the price at that point.

Printing the Rent Visualizer

You can print the Rent Visualizer, if you want to. This will include the graph and the information in the panel on the right side of the screen at the time the Rent Visualizer was printed.

To print the Rent Visualizer, click the Print icon in the top right corner of the screen.

The graph will print on one page and the panel information on another.

Security

Access to the Rent Visualizer is controlled by a property right called Field Rent Visualizer. Users that have this right will be able to see the links on the New Pricing screen and to launch the Rent Visualizer.

This right is included in the following rights::

  • Administrator
  • PricingAnalyst
  • LRO_Administrator
  • LRO_PricingAnalyst

Turn On/Off

You can turn the Rent Visualizer on and off using parameter 313. You may want to turn the Rent Visualizer off if you are not using it or if you find that it slows down your nightly processes too much.

Lease Expiration Management (LEM) must also be turned on to use the Rent Visualizer. LEM is turned on using parameter 172.
