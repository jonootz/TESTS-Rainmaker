Lease Expiration Management Graph

Understanding the Expiration Management Graph

This is the key information you will see on the graph for the Unit Category you have selected.

Actual Expirations (blue): The number of expirations expected from leases which have already been signed.

Remaining Forecasted Expirations (purple): The number of expirations expected from demand which is forecasted to materialize in the future.

Total Forecasted Expirations (blue + purple): The sum of Actual Expirations and Remaining Forecasted Expirations.

Recommended Expirations (red): The expirations recommended for each calendar month based on seasonality, historical expirations, and renewal fractions.

Tips

  • The Expiration Management Graph is broken down by unit category.
  • Select Display Graph Values to show the actual values associated with each month.
  • The scale on the left margin provides the number of expirations as well as the 1.0 average demand for seasonality.
  • Expiration data for 24 months can be viewed using the scroll bar at the bottom of the graph.
  • Seasonality is illustrated as a line near the top of the graph and numerically below the graph.
  • The recommended expirations are illustrated as red bars on the graph and numerically below the graph.
  • The blue bars indicate the actual number of leases in place.
  • The purple bars indicate the remaining forecasted expirations.
  • The green highlighted cells below the graph allow you to override the recommended number of expirations.
