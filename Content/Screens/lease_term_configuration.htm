Lease Term Configuration Screen

The Lease Term Configuration screen allows you to configure your lease terms to offer different lease terms for new leases vs. renewal leases. You can also offer different lease terms for your rent control units, if you have any. This isn't where you set up your lease terms. It is just where you specify which lease terms you want to use for each lease type. Lease terms are set up under LRO configuration or RecGen.

Using the Lease Term Configuration Screen

The Lease Term Configuration screen is located under Admin>Lease Term Configuration.

Lease Term Configuration determines which lease terms will be displayed in 's New Pricing and Renewal Pricing screens. Pricing for the specified lease terms will be sent to your PMS system if you are using Yardi, AMSI, or MRI. It will not change the lease terms you have set up in your PMS system.

Configuring Lease Terms

When you configure lease terms, you have the option to configure the lease terms by lease term or by community.

Configuring a Lease Term for all Communities

To configure a lease term for all communities that use that lease term:

  1. Check the Configure by Lease Term checkbox.

  2. Select the lease term from the Lease Term drop-list.

  3. Update the settings for the lease term for each community listed.

    Click on a cell to change the value from Yes to No or No to Yes. Lease terms that are included are shown in green and lease terms that are excluded are shown in red. Any changes you make will appear in red italics until saved.

    You can easily set the lease term to the same setting for all communities using the "Set All" option.

  4. After you make a change, the Save and Reset buttons appear.

  5. To save your changes, click Save. After you have saved your changes, the buttons will disappear and all of the values will appear in regular black text.

    If you don't want to save your changes, click Reset to set the values back to the last saved values.

Using Set All

On the Lease Term View (Configure by Lease Term checkbox is checked), there is a Set All option for each pricing column. This is on the first row where it says "All" under Community.

If you click Set All, it will change the lease term setting in that pricing column for the specified lease term to No for all communities in the list. It will also show No in the cell for All (instead of Set All).

If you click No, it will change the lease term setting for the specified lease term to Yes for all communities in the list. It will also show Yes in the cell for All.

If you change the lease term setting for one of the communities, then the cell for All will return to Set All since all of the communities in the list no longer have the same setting.

When you first open the screen or if you come back to this screen after saving your changes, the All row will display Set All even if all of the communities are set the same.

Configuring Lease Terms for a Specific Community

To configure the lease terms for a specific community:

  1. Make sure the Configure by Lease Term checkbox is not checked.

  2. In the Community drop-list, select the community that you want to configure the lease terms for.

    You can set different lease terms for affordable new and renewal leases, as well as conventional leases. You'll see columns for both the conventional lease terms and the affordable lease terms on this screen if the selected community has affordable (rent control) units.

  3. Click on a cell to change the value from Yes to No or No to Yes. Lease terms that are included are shown in green and lease terms that are excluded are shown in red. Any changes you make will appear in red italics until saved.

    You must have at least one lease term marked as Yes for New Leases and for Renewals.

  4. After you make a change, the Save and Reset buttons appear.

  5. To save your changes, click Save. After you have saved your changes, the buttons will disappear and all of the values will appear in regular black text.

    If you don't want to save your changes, click Reset to set the values back to the last saved values.

When Will Changes Take Effect

After changes are saved, the lease term changes for New Pricing will be effective the next time pricing is run. Lease term changes for renewal pricing can be seen by refreshing the Renewal Pricing screen.
