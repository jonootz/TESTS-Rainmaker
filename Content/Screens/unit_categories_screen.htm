Unit Categories

In the Web UI, go to Configuration>Unit Categories to set up the unit categories for your unit types.

Unit Categories Screen with Editing Enabled

Understanding the Unit Categories List

Unit Category - This is the name or ID of the unit category. This is entered when the unit category is added and cannot be changed.

Description - This is entered when the unit category is added. You can change it if you want to.

Optimize - This setting determines whether or not prices are calculated for unit types in this unit category. When new unit categories are created, this is set to Yes by default which means prices will be calculated for unit types in this unit category. You may want to set this to No if the unit types in this unit category are rent controlled or have some constraint that makes pricing inapplicable.

Unit Types - These are the unit types that have been assigned to the unit category. Unit types are assigned to unit categories on the Unit Types screen.

Sort Order - This determines the order that the categories appear in the New Pricing report under the tab and possibly other reports. It does not affect the sort order of the list. You can sort the list by clicking on the column heading that you want to sort by.

Action Column - The last column contains the icons for actions you can take on each unit category. Options are Edit , Delete , Save , and Cancel . Save and Cancel are available after you click Edit. Delete is available if the unit category is not assigned to any unit types and is not the default unit category (0x0). You must enable editing for these options to appear in the column.

Understanding Other Unit Category Options

Add Unit Category - Opens a dialog box where you can add a unit category name and description to create a new unit category. You must enable editing to see this option.

Export to Excel - Select the Excel icon in the upper right corner of the screen to open the unit category list in Excel.
