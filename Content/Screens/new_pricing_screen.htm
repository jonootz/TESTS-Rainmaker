New Pricing Report

The New Pricing report shows the pricing for each apartment that is available to rent at the community, grouped by unit type.

Using the New Pricing Report

This report is used by leasing consultants when working with prospective residents. The leasing consultant will use the summary view first to narrow down the list of available apartments. Once an apartment is selected, the detail view can be used if the prospective resident would like a different lease term than the one shown on the summary view.

When working with prospective residents, try to understand what they are really looking for before quoting a price. Be sure they understand that pricing changes daily.

Some items to consider are:

Which floor plan are they interested in?

When do they want to move in?

Is there a particular floor level they are interested in? (i.e., ground floor, top floor)

Would they like a particular view or does it matter?

Are they looking for a particular lease term other than the standard 12 month lease term?

Viewing the New Pricing Report

You'll find the New Pricing report under LRO>New Pricing.

There is also a that has a graphical view of the new pricing and can be run for multiple communities.

The Summary view of the report shows one line per apartment with the best lease term information.

The Detail view of the report shows all lease terms for each apartment. You will use this view to offer flexible lease terms to prospective residents.

Understanding the Information on the New Pricing Report

Unit Type (Unit Cat): Available apartments are grouped by unit type.

Unit ID: Identifies the apartment that is available.

Status: Vacant Available - unit is ready to move in; Vacant Unavailable - unit is empty, but not yet ready to move in; On Notice - current resident is moving out at a future date

Available Date: When the apartment is going to be ready for a new resident to move in.

Move-out Date: When the previous resident of the apartment is moving out.

Prior Rent: This is the rent amount that the previous resident was paying.

Sq Ft: This is the square footage measurement for this apartment. This column is hidden by default. To include it on the report, click the Options button and select Show Sqft.

Amenities: What's special about this apartment - pool view, floor level, renovations, stainless steel appliances, hardwood floors, etc. In parentheses, you'll see the PMS unit type and pricing offset amount, if any.

Amen Amt: This is the cumulative dollar amount of the amenities in the apartment. It is not uncommon to see $5, $20, $50, or more charged for different amenities.

Lease Term: In Summary view, this is the best/cheapest lease term for the apartment. In Detail view, each lease term is shown on its own line.

Base Rent: The price before any concessions.

Total Con: Any concessions that are being offered. A concession strategy can be set up in under Configuration>Recommendation Strategy. You can set up a concession strategy for each unit type and lease term. The concession strategy is a marketing strategy. For example, you may offer a free month's rent or a flat dollar amount as a concession.

Eff Rent: This is the final rent. It will match the base rent unless a concession is being offered.
