Reference Rent

Reference rents are used as a starting point for pricing in the system. The reference rent is set at the unit type level and is the equilibrium point at which a higher price will decrease demand and a lower price will increase demand.

How does the system use the reference rent?

Prices are optimized around the reference rent and may be higher or lower than the reference rent.

If the system is trying to increase demand (get more traffic/leases), then the price will likely be lower than the reference rent.

If there is limited availability and future demand is expected to be strong, then the system may price higher than the reference rent (because there is already more than enough demand to cover the apartments that are available).

Where does the reference rent come from?

The system uses the DRRC (Dynamic Reference Rent Calculator) to set the reference rent.
