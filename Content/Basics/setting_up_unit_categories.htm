Setting Up Unit Categories

Unit Categories are used to categorize your unit types. For example, you may have a unit category for 1 bedroom/1 bathroom units and have multiple unit types that would fall into this category.

You will typically add or delete unit categories during the initial setup of a community. There may be circumstances such as significant rehabs or new inventory coming online when you may want to regroup the unit types into new unit categories.

Learn about the Unit Categories screen

To set up and maintain unit categories:

  1. Log into the Web UI and select the Configuration tab in the top title bar.

  2. Select the community.
  3. Select the Unit Categories screen. This is where you'll set up and maintain the unit categories for the community.

What is the Default Unit Category

When the community is created, a default unit category is set up. It's named 0x0 with Optimize set to No. This means pricing will not be generated for any units in unit types assigned to this unit category. By default, all unit types are assigned to this unit category. You will need to set up new unit categories for the community and assign unit types to them on the Unit Types screen.

You cannot delete the default unit category. It is used by the system whenever new unit types are added.

Adding a Unit Category

  1. On the Unit Categories screen, click Add Unit Category.
  2. Enter a name or ID for the unit category and a description.
  3. Click OK.

Editing a Unit Category

After you've created a unit category, you can edit the description, optimize setting, and sort order.

  1. On the Unit Categories screen, click the pencil icon on the row for the unit category you want to edit.
  2. Make your changes.
  3. Click the save icon on the unit category row to save your changes. If you don't want to save your changes, click the cancel icon .

Deleting a Unit Category

You can delete unit categories that don't have any unit types assigned to them. You cannot delete the default unit category (0x0).

To delete a unit category, click the delete icon on the row of the unit category you want to delete. This icon appears next to the pencil icon. If you don't see the delete icon, then the unit category cannot be deleted.

Renaming a Unit Category

You can't edit a unit category's name. You have to create a new unit category and delete the old one.

You can't do this for the default unit category (0x0) since it can't be deleted, but you can change its description.

  1. On the Unit Categories screen, add a new unit category with the name you want.
  2. Set the unit category up the same way as the original unit category.
  3. On the Unit Types screen, change any unit types that were assigned to the original unit category to be assigned to the new unit category.
  4. On the Unit Categories screen, click the delete icon on the original unit category row. This icon appears next to the pencil icon if the unit category can be deleted.
