Renewal Letters

Your renewal pricing is typically run and renewal letters generated once a month. This is done from the Renewal Pricing report.

Review Renewal Offers

You can print the Management Report from the Renewal Pricing report. This report can be used for management to review and approve the renewal rates. You should save a copy of this dated and approved report.

You can use the Excel icon in the top right corner of the Renewal Pricing report to get a detailed renewal worksheet in Excel, if you would prefer to use that.

Generate Renewal Letters

After the rates have been approved, you should generate and send out the renewal letters. Since the system integrates with your PMS system, you may do this from your PMS system. You can also use the Mail Merge also use the Mail Merge button on the Renewal Pricing report to export a spreadsheet to Excel that can be used with Microsoft Word's mail merge functionality to create the renewal letters. You can also run mail merge renewal letters from the Reporting tab which allows you to generate the mail merge report for multiple communities.

You should save copies of your renewal letters in your computer system and/or in the resident file folders.
