Mapping Competitor Unit Types

After you add a competitor, you will need to verify the unit type mappings. When a new competitor is added, competitor unit types are automatically created and mapped based on your unit types. You can change these or add different ones, if necessary.

Verifying and Editing a Unit Type Mapping

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click the pencil icon on the row for the competitor whose unit type mapping you want to change.
  3. In the Unit Types section, click the row for the unit type mapping you want to verify.
  4. Make sure the name, description, and square footage information are correct. Also, verify the bed and bath information under Automated Comp Settings. If Automated Comps is turned on, make sure the Base Rent is shown which means that a rent was found. If a rent is not found, make sure the bed, bath, and square footage range is correct.
  5. If you have made any changes, click Save. Otherwise, click Cancel.
  6. Click Close.

Adding a New Unit Type Mapping

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click the pencil icon on the row for the competitor whose unit types you want to map to your unit types.
  3. Click the Add New button in the upper right corner of the Unit Types section.
  4. Enter a name for the competitor's unit type, for example, 1X1.
  5. Enter square foot information for the competitor unit type in the SF field.
  6. In the Automated Comp Settings section, select the bed and bath information for the unit type. This is required even if you don't currently use Automated Comps.
  7. If Automated Comps is turned on, make sure the Base Rent is shown which means that a rent was found. If a rent is not found, make sure the bed, bath, and square footage range is correct.

  8. Enter any additional information about the competitor unit type such as a description or number of units.

    Leave Comp Rent Lease Term Category set to the default value, L-Long-Term Lease.

  9. In the My Unit Type Mapping section, select the unit type that is similar to the competitor unit type. Each of your unit types can only be mapped to one competitor unit type.

  10. Click Save.

    Repeat steps 3 - 7 for each of the competitor's unit types you want to add.

  11. When you are finished adding new unit type mappings, click Close.

Deleting a Unit Type Mapping

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click the pencil icon on the row for the competitor whose unit type mapping you want to delete.
  3. In the Unit Types section, click the row for the unit type mapping you want to change.
  4. Click the delete icon on the lower left hand corner of the Competitor Unit Type screen.

  5. If this competitor unit type is mapped to unit types in other communities, you will get a message that this unit type is shared.

    Click Yes to unmap the unit type for this community. It will remain in the system and retain its mapping to unit types in other communities.

    If this competitor unit type is not mapped to unit types in other communities, you will get a message giving you the option to unmap the unit type, but leave it in the system or to delete the unit type out of the system.

    Click Yes to delete the unit type from the system. Click No to unmap the unit type, but leave it in the system so you can map it to another unit type in the future.

  6. Click Close.
