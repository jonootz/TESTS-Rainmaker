Overriding Expirations

When Lease Expiration Management is enabled, the system will recommend the desired expirations for each calendar month. However, you may feel that you want to allow more or fewer expirations in a given month. For that reason, you can override the system recommended expirations for a particular month.

To do this:

  1. Go to LRO>Expiration Management>Expiration Management Graph.

  2. Below the graph, you'll see the numeric values of the system recommended expirations for each month.

    Enter the override values in the green cells below the recommended values.

  3. Click Save.
  4. Click Re-Optimize to see the effect the overrides have on new lease pricing.

Things to Keep in Mind When Entering Expiration Overrides

  • The override value entered represents the total number of lease expirations preferred in a given month.
  • The override value cannot exceed 15% of the total units in the unit category. This maximum allowable percentage is set using parameter 226 which is set to 15 by default.
  • To preserve the recommended expiration counts totaled over the year, the Recommended Total and Overridden Recommendation Total should match.
  • Once an override is saved, it replaces the expiration recommendation for the month in the current year and in future years (unless you change it in the future).
  • After the overrides have been entered, the values should be saved and then pricing can be re-optimized.
  • Overrides will be reflected on the graph in orange.
