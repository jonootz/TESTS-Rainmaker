Mapping Unit Types

PMS unit types must be mapped to an unit type. This is done automatically when the community is set up or when new unit types are imported from the PMS system. You can see the PMS unit types and the unit types they are mapped to in the Web UI under Configuration>PMS Unit Types.

Learn about the PMS Unit Types screen

Why You Would Change The Mapping

  • You created a new unit type that you want to use.
  • A new PMS unit type was imported into the system and you want to map it to a different unit type instead of accepting the default 1-to-1 mapping.
  • A PMS unit type was mapped to the wrong unit type.

To view the PMS unit types and change the unit type mappings:

  1. Log into the Web UI and select the Configuration tab in the top title bar.

  2. Select the community.
  3. Select the PMS Unit Types screen. This is where you'll view and map the PMS unit types for the community.

  4. If you want to change the mappings, click Enable Editing if the community is not locked.

Mapping a Unit Type

  1. On the PMS Unit Types screen, select the pencil icon on the row for the PMS unit type you want to map.

  2. In the Unit Type field, select the unit type from the drop list.

    If you don't see the unit type, go to the Unit Types screen and create it.

  3. Click the Save icon. The PMS unit type is now mapped to the unit type you selected.
