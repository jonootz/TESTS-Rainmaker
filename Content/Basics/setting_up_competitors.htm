Setting Up Competitors

Competitors are the communities in the area that your community competes with for traffic and leases. Ideally, you'll enter 4-6 competitors for your community. You will typically set up your competitors when you are initially setting up the community in the system. Then you will enter competitor rents each week on the Comp Rents screen. You may need to periodically edit or delete competitors or add new ones, depending on what is happening in the area near your community.

Competitors are set up under LRO>Competitors>Comp Setup. This is where you enter information about your competitors and map their floorplans to unit types.

After setting up your competitors, you'll need to enter competitor weights and positions.

Adding a Competitor

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click Add Competitor.

  3. Potential competitors and existing competitors will appear on the map and in a list on the left side of the screen.

    Competitors that are already set up will appear in the list with a green checkmark.

    Select each community that you want to use as a competitor. You should typically select an Automated Comps community, whether you are using Automated Comps or not. See tips for using the map

    If you don't see a competitor that you are looking for, you can add it manually.

  4. Click the Add button.

    If your community is set up to use Automated Comps, when you add an Automated Comps competitor, it will be set to use Automated Comps.

  5. On the Comp Setup screen, click the pencil icon on the row for the competitor you just added.
  6. Verify the competitor information is correct. If not, click the Edit button for that section and change it. You can also enter information about why this community is a competitor and the competitor's utilities, amenities, and pet policies.
  7. Verify the unit type information and unit type mappings are correct.
  8. If you are using Automated Comps, make sure the unit types are set up correctly for Automated Comps, as well.
  9. Repeat steps 5-8 for each competitor that you added.
  10. Click Close.

Tips for Using the Map to Add Competitors

Zooming on the Map

You can zoom in and out on the map to see fewer or more communities.

If there are multiple communities in the same area, you may see an icon with a number which represents the number of communities in that area. If you click on the icon, it will zoom to that area to show you the communities.

If you zoom out too far, the map will stop filtering and searching. You'll need to zoom in some if this happens.

Understanding the Properties Noted on the Map

The legend at the top of the map will help you keep track of the communities that are pinned.

You can click on the Auto Comp, Sister, or Manual pin in the legend to only display those communities on the map.

Red pin - This is your community. You can see where it is on the map in relation to other communities.

Dark Blue Pin - These are the communities that have at least one available floorplan and are eligible to use Automated Comps to get rents. When adding a new competitor, you would typically select one of these communities, whether you use Automated Comps or not.

Light Blue Pin - These are sister communities. Sister communities are other communities that you have in . If you choose one of these communities as a competitor, the generated price will be used as the comp rent. You cannot use Automated Comps with sister communities.

Blue Pin - These are the competitors that have been entered manually for your community or another community. They are not set up to use Automated Comps.

Green Pin (not in the legend) - These are the competitors that you have selected to add, but haven't added yet.

Available Floorplans

If you are using Automated Comps, when you click on an Auto Comp competitor pin (Dark Blue), you can see which floorplans are currently available at that property.

Reasons You May Not See A Competitor on the Map

If a competitor does not appear on the map, it is not available to use Automated Comps and you will need to manually enter it. A competitor may not be available to use Automated Comps because:

  • quality rent information is not available for the property
  • address information is wrong

Automated Comps is not available for properties in Canada.

Reasons You May See A Competitor on the Map Multiple Times

  • A competitor may appear on the map multiple times because the address is slightly different. For example, you may have the competitor manually entered with the address as 112 S. Main St., Automated Comps may have the address as 112 South Main Street, and another community may have the same competitor with an address of 112 Main Street, South.
  • You may also see a competitor on the map multiple times if the same competitor has been saved manually for different communities.

If you are using Automated Comps, you should transition your existing competitors to Automated Comps before adding new ones. This way, you'll be less likely to see duplicates.

When you are setting up a new competitor, if you see multiple instances of a competitor and aren't sure which one to choose, you should select the Automated Comps one, even if you aren't using Automated Comps.

Selecting Competitors to Add

You can select the competitor or competitors that you want to add by selecting the name from the list or by clicking the pin on the map, then clicking Select.

You can unselect a competitor by clicking on it in the list or by clicking the pin on the map, then clicking Deselect.

If there is a long list of communities, you can use the Search box at the top of the list to find the one you want. This searches through the communities that are listed.

You can select the Selected Comps button to see a list of comps you have selected to quickly review them and unselect any of them, if necessary.

Editing a Competitor

  1. Go to LRO>Competitors>Comp Setup.

  2. On the Comp Setup screen, click the pencil icon on the row for the competitor you want to edit.

    There are four sections of information which you can edit: Competitor Information, Competitive Information, Utilities/Amenities/Pets Information, and Unit Types.

  3. Click the Edit button in the upper right corner of the section containing the information you want to change.

    If you want to edit the Unit Type information, click the row of the unit type you want to edit or click Add New if you want to add and map a new competitor unit type.

  4. Make your changes.

  5. Click Save.

    Repeat steps 3 - 5 for each section that you want to edit.

  6. After you have made all of your changes, click Close.

Deleting a Competitor

  1. Go to LRO>Competitors>Comp Setup.
  2. On the Comp Setup screen, click the delete icon on the row for the competitor you want to delete.

  3. If this competitor is used by other communities, you will get a message that this competitor is shared.

    Click Yes to detach the competitor from this community. It will still remain in the system as a competitor for other communities.

    If this competitor unit type is used by other communities, you will get a message giving you the option to detach from this competitor, but leave it in the system or to delete the competitor out of the system.

    Click Yes to delete the competitor from the system. Click No to detach the competitor from this community, but leave it in the system for future use.
