Dynamic Reference Rent Calculator (DRRC)

As its name suggests, the Dynamic Reference Rent Calculator calculates the reference rent. It is considered "dynamic" since the reference rent changes based on community performance.

The DRRC updates the reference rent once a week, by default. This is typically done on the day of the pricing call. This day is specified using parameter 222 when the system is set up, but can be changed if the day of your pricing call changes.

The DRRC takes into account Exposure (supply) and Leasing Velocity (demand) when calculating the reference rent.

Why is the reference rent only updated once a week?

When the reference rent is updated, there is the potential for a larger change in price than on a regular day. Allowing the reference rent to stay the same for a week results in pricing that stays around the same point for the week. This allows sufficient time to test pricing at the community since traffic and leasing is not always consistent from day to day. For example, you may have more traffic on the weekend than you do on a weekday.

The reference rent can be updated more than once a week, but we recommend updating it once a week.
