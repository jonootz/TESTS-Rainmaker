When and How to Move from Seeded to Calculated Statistics

Seeding statistics is when you tell the system to use data consistent with assets in the same market instead of calculating statistics using data from the Property Management System.

If necessary, support can help you make the move from seeded to calculated statistics.

When to Move from Seeded to Calculated Statistics

You would use seeded statistics when you don't have enough good data for the system to calculate valid statistics. Following are guidelines for the amount of lease history you need to move from seeded to calculated statistics along with the parameter that controls whether or not that statistic is seeded or calculated.

How to Move from Seeded to Calculated Statistics

This is what you will need to do when you are ready to move from seeded to calculated statistics. Contact Support, if you need assistance.

  1. Go to Admin>Parameters.

    You should be on the Community Parameters screen. If not, select Community under Parameters in the menu bar.

  2. Make sure the Community you want to change the parameters for is selected.
  3. Click Edit in the upper right corner of the screen.
  4. Scroll down to find the parameter for the statistic that you want to be calculated.
  5. In the green field (Community Value) for that parameter, enter 1. (0=Seeded, 1=Calculated)
  6. Repeat steps 4-5 for each parameter you want to change.
  7. Scroll down to parameter 223 Upstat DOW Run Date and make a note of what this parameter is currently set to.
  8. After you have noted what parameter 223 is set to, change it to the current day of the week (0=Sunday, 1=Monday, etc.).
  9. Click Save.
  10. Go to Admin>Scheduler.
  11. Click Nightly.
  12. Check the box next to the property whose parameters you just changed.
  13. Click OK to run a fully nightly processing. This will allow the calculated statistics to take effect.

  14. After the nightly processing has completed, go to Admin>Parameters.

    You should be on the Community Parameters screen. If not, select Community under Parameters in the menu bar.

  15. Make sure the Community whose parameters you changed is selected.
  16. Click Edit in the upper right corner of the screen.
  17. Find parameter 223 and change it back to what it was previously.
  18. Click Save.

If you want to review what the calculated statistics look like, go to LRO>Charts>BI Charts. Use the Report drop-down to select the different statistics graphs.
