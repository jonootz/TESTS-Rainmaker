Lease Expiration Management (LEM) Parameters

The following parameters are used when parameter 172 is set to 1.

The community level historical data will be used to determine expiration counts for a unit category if it has less than the number of units set in parameter 173.

173 LEM Minimum Number of Units in UC

Parameter 174 is used to smooth prices from one day to another to reduce volatility. This parameter is used when parameter 132 is set to A. Selecting A for parameter 132 means that you want to smooth over all weeks when using LEM. Parameter 174 is the weight you want to give to yesterday's price when smoothing.

174 FO Smoothing Weight

132 Smoothing Option with Rec or Current Rent

Parameter 178 can be used to restrict the recommended prices to a maximum percentage over the reference rent.

178 FO Maximum Price Increase

You can use parameter 181 to control whether or not to use remaining expirations (purple bars) on the LEM graph and in calculations.

181 Use Remaining Expirations (LEM)
