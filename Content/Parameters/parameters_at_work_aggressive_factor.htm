Parameters at Work: Aggressive Factor

Aggressive Factor Parameters

Aggressive Factor (#73)

This parameter is set as a percentage. It should be a small number such as 2%.

Purpose

This parameter will affect Reference Rent which you can see on the Rent Comparison graph of the Pricing Trend Report. The system increases the reference rent by the aggressive factor to test increases in pricing when exposure is low and there is some leasing activity. As long as exposure is low and there is some leasing activity, the Aggressive Factor will apply each week when the DRRC updates the reference rent.

In the above screenshot, the aggressive factor has come into play on 11/26, 12/3, 12/10, 12/24, 12/31, and 1/7. The reference rent was increased by 2% on each of those dates in response to low exposure with leasing activity.

Example

Let's say you have parameter #73 Aggressive Factor set to 2%. If your exposure is low and there is some leasing activity, when the DRRC is run this week, the LRO system will increase the reference rent by 2% to test pricing at a slightly higher level. If the community continues to have low exposure with leasing activity (pricing power), the system will increase the reference rent by another 2% the next time the DRRC is run (typically the next week).
