Parameters at Work: Min Increase and Max Increase

Min Increase and Max Increase

The Min Increase and Max Increase parameters have a significant effect on the calculation of renewal offers. These parameters are set on the Renewal Config Parameters screen which you can find at Admin>Parameters>Renewal Config.

These parameters are entered in the Min Increase and Max Increase fields.

Min Increase

Purpose

Min Increase sets a minimum percentage increase of the resident's current rent. In an inverted lease situation (new move-in rent is less than what the resident is currently paying), the Min Increase sets a cap on how much the resident's current rent can be decreased. If this is set to 0%, it means no decrease in the current rent is allowed.

Example

Increase Rate: 25%

Min Increase: 5%

Resident's Current Rent: $1500

New Move-In Rent $1700

Difference between Resident's Current Rent and the New Move-In Rent: $200 ($1700 - $1500)

Renewal Increase: $50 ($200 X 25%)

Min Increase Amount: $75 ($1500 X 5%)

Renewal Offer: $1575 ($1500 + $75)

Although the renewal increase is $50 based on the Increase Rate, the minimum increase is $75.

 

Example of the Min Increase in an Inverted Lease Situation:

Drop Rate: 30%

Min Increase: 5%

Resident's Current Rent: $1500

New Move-In Rent $1200

Difference between Resident's Current Rent and the New Move-In Rent: $300 ($1500 - $1200)

Renewal Decrease: $90 ($300 X 30%)

Min Increase Amount: $75 ($1500 X 5%)

Renewal Offer: $1425 ($1500 - $75)

Although the renewal decrease is $90 based on the Drop Rate, the cap on the decrease in rent is $75.

Max Increase

Purpose

Max Increase limits the amount that a resident's rent can increase. This is a percentage of the resident's current rent and serves as a cap to the Increase Rate. If this is set to 0%, it means no increase in the current rent is allowed.

The Max Increase does not come into play in the case of an inverted lease (new move-in rate is less than what the current resident is paying). It also does not apply to premium leases (short term and MTM leases).

Example

Increase Rate: 70%

Max Increase: 5%

Resident's Current Rent: $1500

New Move-In Rent $1800

Difference between Resident's Current Rent and the New Move-In Rent: $300 ($1800 - $1500)

Renewal Increase: $210 ($300 X 70%)

Max Increase Amount: $75 ($1500 X 5%)

Renewal Offer: $1575 ($1500 + $75)

Although the renewal increase is $210 based on the Increase Rate, it is limited by the Max Increase parameter to $75.
