Parameters for Seeding Statistics

Seeding statistics is when you tell the system to use data consistent with assets in the same market instead of calculating statistics using data from the Property Management System.

You would use seeded statistics when you don't have enough good data for the system to calculate valid statistics. You should consider using calculated statistics once you have 12 months of good data. For Seasonality, you should have 18 months of good data.

If necessary, support can help you make the move from seeded to calculated statistics.

For the following parameters, 0=seeded and 1=calculated. The default is 1.
