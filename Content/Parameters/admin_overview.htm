Scheduling Processes

You can set the day of the week that you want your pricing processes to run.

Parameter 222 controls the day of the week that the LRO system will run the DRRC process which sets a new reference rent.

Parameter 223 controls the day of the week that the LRO system will run the UPSTAT process which calculates a full set of community statistics.

You would typically run both of these processes on the same day as the pricing call. This way, the processes run in the early morning and will factor into that day's pricing. The biggest changes can occur when the DRRC runs so timing the pricing call with the DRRC run means you can review the results that same day.

These parameters are usually set when you first start using the LRO system. You would change them if you move your pricing call to a different day and you want the DRRC and UPSTAT to continue to run on the same day as the call.

222 DRRC DOW Run Date

223 Upstat DOW Run Date
