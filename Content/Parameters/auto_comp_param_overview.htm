Automated Comps Parameters

Automated Comps makes it easy to enter competitor rents each week. It automatically gets the rent information for the competitors that have been set up to use Automated Comps.

Parameter 317 EnableAutoComp is used to turn Automated Comps on at the community level.

Comp Rents

Automated Comps will get new comp rents each week. Parameter 320 determines when Automated Comps will get the rents each week. Parameter 318 specifies when an alert should appear on the Comp Rents screen if Automated Comps doesn't get a rent.

An alert will appear on the Comp Rents screen if the rent changes by more than the amount specified by parameter 112. Parameter 319 specifies the maximum amount that the comp rent is allowed to change when received from Automated Comps.

320 Days to shop Auto Comp prior to DRRC

318 MissingWeekThreshold

112 Rent Low Change % Threshold

319 Max Rent Change % Threshold

These parameters will not be visible unless Automated Comps is turned on.
